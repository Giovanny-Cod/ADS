# Projeto de Inclusão Digital - ADS

## Descrição do Projeto
Este projeto foi desenvolvido como parte das Atividades Extensionistas do curso de Análise e Desenvolvimento de Sistemas. O objetivo é a democratização do acesso a conteúdos educacionais digitais através de uma plataforma web.

## Setor de Aplicação
Escola pública municipal da cidade de Almirante Tamandaré – PR. O projeto foi estruturado para atender às necessidades de suporte digital no laboratório de informática da instituição.

## Tecnologias Utilizadas
- **Linguagem**: JavaScript
- **Framework**: React
- **Banco de Dados**: Supabase

## Objetivos
1. Desenvolver uma plataforma web educacional para centralizar conteúdos de alfabetização digital.
2. Implementar um sistema de banco de dados em tempo real para gestão eficiente de materiais didáticos.
3. Promover a inclusão digital através de uma ferramenta acessível e funcional.

## Como Executar
1. Clone este repositório.
2. No terminal, execute `npm install` para instalar as dependências.
3. Configure suas chaves de ambiente do Supabase.
4. Execute `npm start` para iniciar a aplicação.
